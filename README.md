# 미션 - 자동차 경주 게임

## 기능 구현 진행 과정
- 경주할 자동차 이름 입력 받기
    - "," 기준으로 입력된 이름 나누기
    - 각각의 이름 입력이 5글자인지 확인하기
    - 중복된 이름 확인하기
    - 참가자 인원수 확인하기

- 시도 횟수 입력받기
    - 입력 값이 숫자인지 확인

- 자동차 경주 진행
    - 참가자 2명 이상인지 확인
    - 시도 횟수만큼 반복
    - 랜덤 숫자를 활용하여 차이동
    - 매번 라운드 상황 표시
    - 최종라운드 종료 후 승자 표시
        - 우승자가 1명인 경우
        - 우승자가 여러명인 경우

- 추가사항
    - 예외 상황인 경우, "[ERROR]" 를 추가한 메세지 제공
